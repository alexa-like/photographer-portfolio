# Photographer Portfolio

A clean, responsive static website for showcasing a photographer's work. Features:

- Hero section with a call‑to‑action
- Dynamic image gallery (vanilla JS)
- About section with a portrait
- Contact form with client‑side validation
- Dark‑mode support via `prefers-color-scheme`
- Mobile‑first layout using Flexbox and CSS Grid

## Project Structure
```
photographer-portfolio/
├─ index.html
├─ style.css
├─ script.js
└─ README.md
```

## Development
1. Clone the repo.
2. Open `index.html` in a browser to view locally.
3. Edit `style.css` or `script.js` to customize images, colors, or content.

## Deployment
Deploy free on Netlify:
- Drag the project folder onto https://app.netlify.com/drop **or**
- Push the repo to GitHub and connect it in Netlify for continuous deployment.

The site is fully static, so any static‑file host (GitHub Pages, Vercel, Cloudflare Pages) works as well.
